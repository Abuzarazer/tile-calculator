# Онлайн-калькулятор плитки — Almali Ceramic

Веб-калькулятор для расчёта количества упаковок, площади, веса и стоимости
керамогранита. Данные о позициях (артикул, название, размер, остаток на складе)
подгружаются **вживую из Google-таблицы** при открытии страницы.

**Живая версия:** https://abuzarazer.github.io/tile-calculator/

## Возможности

- Загрузка каталога из Google Sheets без бэкенда (JSONP через Google Visualization API — обходит CORS даже при открытии файла локально).
- «Умный» поиск столбцов: скрипт сам находит нужные колонки по ключевым словам в заголовках, поэтому переименование/перестановка столбцов в таблице не ломает калькулятор.
- Поиск по артикулу / названию / размеру с выпадающим списком.
- Расчёт: упаковок (округление вверх), площади к покупке, штук, веса, итоговой стоимости.
- Предупреждение, если остатка на складе не хватает на нужное число упаковок.
- Адаптивная вёрстка + отдельная поддержка мобильного Safari (iOS/iPadOS).

## Структура

Проект — **один самодостаточный файл**: `index.html` (HTML + CSS + JS, без зависимостей и сборки).

## Как запустить

- **Локально:** открыть `index.html` двойным кликом в Chrome/Edge/Safari (desktop). JSONP-загрузка из таблицы работает и с `file://`.
  ⚠️ В мобильном Safari локальный `file://` часто не исполняет JS — для телефона используйте ссылку хостинга (см. ниже).
- **По ссылке:** хостится на GitHub Pages из ветки `main` (корень репозитория).

## Деплой

Хостинг — GitHub Pages, источник: ветка `main`, папка `/`.
Любой push в `main` автоматически пересобирает сайт (~1 минута):

```bash
git add index.html
git commit -m "..."
git push
```

Перед публикацией GitHub Actions проверяет обязательные элементы страницы и синтаксис встроенного JavaScript. Сценарии ручной проверки на телефонах и планшетах — в [MOBILE_TESTING.md](MOBILE_TESTING.md).

Данные о плитке править **не в коде, а в самой Google-таблице** — калькулятор подтянет их при следующем открытии.

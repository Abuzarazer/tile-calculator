# CLAUDE.md — контекст проекта для Claude Code

Онлайн-калькулятор плитки (Almali Ceramic). Один файл `index.html` (HTML+CSS+JS,
без сборки и зависимостей). Ниже — всё, что нужно знать, чтобы безопасно вносить правки.

## Что это

Калькулятор считает по выбранной плитке: упаковки, площадь к покупке, штуки, вес,
стоимость; предупреждает о нехватке на складе. Каталог грузится вживую из Google Sheets.

- Живой сайт: https://abuzarazer.github.io/tile-calculator/
- Хостинг: GitHub Pages, ветка `main`, папка `/`. Push в `main` → пересборка ~1 мин.

## Источник данных

- Google Sheets по JSONP через Google Visualization API (обходит CORS, работает и с `file://`):
  `https://docs.google.com/spreadsheets/d/{SHEET_ID}/gviz/tq?tqx=out:json;responseHandler:{callback}&sheet={sheetName}`
- `SHEET_ID = 1dqYgo6PI2ttiQgbF8QhLzOI19VSFSEdthK_xDX5Mk04`
- Перебираются листы: `['60*60','60*120','60x60','60x120','60х60','60х120']`.
  Фактически в таблице реальные вкладки названы иначе, поэтому все запросы отдают
  один и тот же лист по умолчанию (60×60). Это НЕ баг — защита от дублей схлопывает
  повторы, на выходе ~79 уникальных позиций.

## Логика загрузки (важно при правках)

1. **Умный поиск колонок** по ключевым словам в заголовках (регистронезависимо, по подстроке):
   - артикул: `артикул`, `артик`; **если не найдено — берётся столбец 0** (в этой таблице
     заголовок артикульной колонки содержит пример артикула, а не слово «артикул»);
   - название: `назв`, `наимен` (заголовок «НАЗВАНИЯ» не содержит подстроку «название» — нужен корень);
   - размер: `формат`, `размер`, `разм`;
   - упаковка: `пакинг`, `упаковк`, `кв.м`, `м2`, `packing`;
   - цена: `цена`, `стоимост`, `прайс` (в текущей таблице колонки цены НЕТ → поле цены
     пустое, менеджер вводит вручную);
   - склад: `москва`; если нет — `склад`, `остаток`. Берётся первый матч, поэтому
     «СКЛАД Москва» выигрывает у «РЕЗЕРВ Москва».
   ⚠️ Не добавляйте в ключи артикула слово `арт.` — оно ошибочно матчит «Статус арт.».
2. **Очистка чисел:** удаляются пробелы (в т.ч. неразрывные), запятая → точка, отбрасывается
   всё кроме цифр/точки/минуса (`1 200` → 1200; `1,44` → 1.44).
3. **Защита от дублей:** уникальный ключ = `Артикул + Размер` (нормализованный).
4. **Нормализация размера:** любые написания (`60x60`, `60х60`, `60*60`, `600x600` мм) →
   каноническое `60x60` / `60x120`. Строки с нераспознанным размером отбрасываются.
5. **Жёсткие параметры (перекрывают таблицу!):**
   - `60x60`: площадь упаковки = **1.8 м²**, штук = **5**, вес = **34 кг**;
   - `60x120`: площадь упаковки = **1.44 м²**, штук = **2**, вес = **27 кг**.

## Правила расчёта

- Упаковок = `Math.ceil(нужная_площадь / площадь_упаковки)`.
- Площадь к покупке = упаковки × площадь_упаковки.
- Штуки/Вес = упаковки × (штук/вес) — только если параметры известны.
- Итого = площадь к покупке × цена (форматируется с пробелами: `150 000 руб`).
- Нехватка склада: если `Math.floor(склад_м² / площадь_упаковки) < нужных упаковок` →
  показать оранжевое предупреждение; иначе — скрыть.

## Соглашения по коду

- Чистый **ES5**, без стрелочных функций, `let/const`, шаблонных строк — ради совместимости
  со старым WebKit/Safari. Не «модернизировать» без причины.
- Функция расчёта глобальная: `window.calculate = function(){ ... }` (вызывается из `onclick`).
- Выбор пункта списка — по событию **`click`** (не `mousedown`).
- Ошибки показывать **в интерфейсе**, никаких `alert()`.
- Есть слой совместимости с iOS Safari: мета-теги, `-webkit-appearance:none` на полях,
  16px шрифт полей (против авто-зума), `-webkit-overflow-scrolling`, safe-area, префиксы градиентов.
- Тема/данные каталога — в коде НЕ хардкодить: каталог только из таблицы.

## Как проверить правку

Нет тестов/сборки. Проверка — вручную:
1. Открыть `index.html` в desktop-браузере, дождаться статуса «Успешно загружено N позиций».
2. Ввести название/артикул → выбрать из списка → задать площадь → «Рассчитать».
3. Проверить крайние случаи: пустой ввод, запятая в площади, нехватка склада.
Логику парсинга можно быстро прогнать в Node на реальном ответе gviz-эндпоинта
(fetch того же URL без `responseHandler`, распарсить `table.cols` / `table.rows`).

## Осторожно

- Мобильный Safari по `file://` часто не исполняет JS — тестировать телефон только по ссылке хостинга.
- Внутри артефакта claude.ai строгий CSP блокирует JSONP — для «живых» данных нужен обычный хостинг (GitHub Pages).
